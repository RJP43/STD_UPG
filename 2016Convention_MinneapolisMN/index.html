Organizing Hierarchies

Important Concepts to Understand when creating Hierarchies:

  • Russian Nesting Dolls
  • Family Tree
  • BOXES in BOXES

Russian Nesting Dolls

Family Tree

  • element hierarchies resemble the descent of a family tree
  • relationships are referred to similarily ie.
    1. ancestor
    2. descendent
    3. sibling
    4. parent
    5. grandparent

Boxes in Boxes

  • Elements are boxes
  • Attributes distinguish box types.

Well-formedness

  • Everything (elements, tags, attributes, references, comments, processing instructions, and marked sections) is properly delimited.
  • There is a single root element that contains all of the other elements both structural and contextual in nature.
  • No elements overlap; therefore, there is a heirarchy or kinship relation in place.

Validity

A valid XML document follows the Rules of a Schema

  1. available vocabulary used to tag elements & attributes
  2. grammar for how the vocabulary is used: rules for nesting, sequencing, etc.
